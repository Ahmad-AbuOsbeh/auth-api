**Setup**
<br>

- .env requirements
- PORT - 3003
- Running the app
- npm start
- Endpoint:

**user data**

```
data should be:
 const obj =  {
   "name":"ahmad",
  "password": 1234,
  "role": "admin"
  };

```

**1- food data**

```
data should be:
 const obj =  {
   "name":"orange",
  "calories": 250,
  "type": "FRUIT"
  };

```

**2- clothes data**

```
data should be:
 const obj =  {
   "name": "T-shirt" ,
  "color": "white" ,
  "size": "M"
};

```

**example hit :**
[sign up](https://ahmad-osbeh-auth-api.herokuapp.com/signup)

- https://ahmad-osbeh-auth-api.herokuapp.com/signup

**example hit :**

- https://ahmad-osbeh-auth-api.herokuapp.com/signin

**example hit :**
[food / clothes](https://ahmad-osbeh-auth-api.herokuapp.com/food)

- https://ahmad-osbeh-auth-api.herokuapp.com/food
